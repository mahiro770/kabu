# 株式AI分析ツール 仕様書

最終更新: 2026-07-18

対になる技術文書として [設計書.md](./設計書.md)（アーキテクチャ・モジュール構成・データ設計）がある。本書は「何ができるか」を利用者・機能単位で記述する。

## 1. 目的・対象範囲

日本株・米国株について、テクニカル指標・財務指標・AIレポート・ニュース発の銘柄候補・条件検索・チャートパターン学習を1つのWebアプリで提供し、個人投資家の銘柄調査を支援する。

- 対象銘柄: `.T`サフィックス付きの日本株全銘柄（東証上場銘柄、yfinanceが対応する範囲）、および米国株全銘柄（同上）
- 対象外: 投資助言・自動売買・注文執行は行わない。すべての出力は参考情報であり、投資判断は利用者の自己責任とする（各画面に免責文言を表示）
- 利用者: URLを知っていれば誰でもアクセス可能（会員登録不要）。ウォッチリストの個人利用・コミュニティ共有のみ、名前＋合言葉による軽量な区分けを設ける

## 2. 画面一覧

| # | 画面 | ファイル | 概要 |
|---|---|---|---|
| 1 | 銘柄分析（ホーム） | `app.py` | 個別銘柄のチャート・指標・財務・AIレポートを表示する中心画面 |
| 2 | 注目銘柄スクリーニング | `pages/1_注目銘柄スクリーニング.py` | 当日のニュース見出しをAIに分析させ、翌営業日以降に動きそうな銘柄を抽出する |
| 3 | ウォッチリスト一覧 | `pages/2_ウォッチリスト一覧.py` | 登録銘柄を現在値・値動きグラフ・メモ・フォルダ付きでまとめて確認する |
| 4 | チャート学習 | `pages/3_チャート学習.py` | テクニカル指標・ローソク足パターンの意味を解説し、該当銘柄を検出する |
| 5 | 使い方 | `pages/4_使い方.py` | アプリ全体の使い方ガイド |
| 6 | 条件スクリーニング | `pages/5_条件スクリーニング.py` | 時価総額・PER等の財務条件で銘柄を検索する |

## 3. 機能仕様

### 3.1 銘柄分析（ホーム画面）

#### 3.1.1 銘柄の指定

| 項目 | 内容 |
|---|---|
| 入力形式 | 証券コード（4桁数字。日本株として`.T`を自動付与）／ティッカーシンボル（英字。米国株として扱う）／会社名の一部（部分一致検索） |
| 解決順序 | ①入力値をそのままティッカーとして株価取得を試行 → ②失敗時は会社名検索でヒットした候補を順に試行 → ③すべて失敗した場合はエラーメッセージと入力例を表示 |
| 実行契機 | 「分析」ボタンのクリック、または他画面からの銘柄クリック（`st.session_state.current_ticker`経由） |

#### 3.1.2 ヘッダー指標

現在値（前日比・騰落率）、日本株の場合は夜間PTS株価と東証終値比（取得できた場合のみ）、52週高値/安値、RSI(14)、年率ボラティリティ、総合シグナルバッジ（買い/やや買い/中立/やや売り/売り）を表示する。

#### 3.1.3 タブ: 価格チャート

- ローソク足チャート（日足、分析期間は3ヶ月〜5年から選択）
- オーバーレイの複数選択: MA20/MA50/MA200・ボリンジャーバンド・一目均衡表（雲は26日先まで投影表示）・VWAP
- テクニカルシグナル一覧: MAクロス・RSI・MACD・ボリンジャーバンド・一目均衡表・ADX・OBV・VWAPの8指標を、指標ごとに買い/売り/中立バッジ付きカードで表示
- 日経平均（日本株）またはS&P500（米国株）との相対パフォーマンス比較チャートと、1ヶ月/3ヶ月/6ヶ月/1年の期間別超過リターン

#### 3.1.4 タブ: インジケーター

RSI・MACD・ADX・OBVのそれぞれについて、専用チャートと直近値の一覧表を表示する。

#### 3.1.5 タブ: 財務指標

| 区分 | 表示内容 |
|---|---|
| バリュエーション | PER（実績/予想）・PBR・EPS・時価総額 |
| 収益性 | ROE・ROA・純利益率・営業利益率・粗利益率 |
| 成長性・財務健全性 | 売上成長率・利益成長率・流動比率・D/Eレシオ・配当利回り・配当性向・自己資本比率 |
| 信用取引（日本株のみ） | 信用買残・信用売残・信用倍率、直近の週次推移（取得元により最新1週分のみの場合あり） |
| 過去の業績推移 | 直近数年分の売上高・営業利益・純利益・自己資本比率 |
| 業績予想 | アナリスト予想EPS・予想売上高・増収増益率・目標株価（平均/高値/安値）・アナリスト評価 |
| 会社概要・役員一覧 | 事業内容（日本語訳）、役員氏名・役職・年齢 |
| 大株主一覧（日本株のみ） | 上位株主の名称・保有比率・保有株数 |
| 関連リンク | 四季報オンライン・Kabutan・バフェット・コード・Yahoo!ファイナンス等の外部サイトへのリンク |

信用取引・大株主は取得元が複数あり（設計書4.1節）、取得できたソース名をキャプションに明示する。全ソース失敗時は「データは現在取得できません」というメッセージと株探への直接リンクを表示し、機能停止に見せない。

#### 3.1.6 タブ: AI分析

「AI分析を実行」ボタン押下で、選択中のAIモデル（サイドバー「設定」で選択）がテクニカル指標・財務指標・シグナルを踏まえたレポートをストリーミング生成する。レポート構成は「現状分析」「テクニカル分析」「短期見通し」「中長期見通し」「具体的な戦略（買い場/売り場/損切りライン）」「注意すべきリスク」「総合評価」の7セクション。応答が得られない場合はエラーメッセージを表示し、アプリ全体は停止しない。

#### 3.1.7 サイドバー: ウォッチリスト

| リスト種別 | 公開範囲 | 認証 |
|---|---|---|
| 🌐公開 | サイト訪問者全員 | なし。誰でも追加・削除可能 |
| 🏘️コミュニティ | コミュニティ名と合言葉を知っている人 | コミュニティ名＋合言葉 |
| 👤個人 | 本人のみ | 名前＋合言葉 |

- 銘柄追加: 証券コード/ティッカー/会社名で検索し、リストに追加
- 各銘柄行: 銘柄名ボタン（クリックでその銘柄の分析画面に遷移）、削除ボタン（✕）、現在値・前日比、日付軸・価格帯付きのミニチャート
- 合言葉紛失時: 「合言葉を忘れた場合はこちら」から、名前のみで合言葉を再設定できる（本人確認なし。既存の登録銘柄は保持される）
- ログアウト: 個人/コミュニティタブで認証済みの場合に「🚪ログアウト」ボタンを表示。押下すると名前・合言葉入力欄と所有権情報をクリアし未認証状態に戻す。ブラウザを閉じた場合も同様にログアウトした状態になる

#### 3.1.8 サイドバー: 設定

- AIモデル選択: 🦙Ollama（ローカル、無料・無制限だがローカル起動が必要）／✨Gemini（クラウド、無料枠あり）／⚡Groq（クラウド、無料枠あり、高速）から、利用可能なものを選択
- 表示言語: 日本語／English（インライン切り替え、財務用語・UI文言が翻訳される）
- 分析期間: 3ヶ月/6ヶ月/1年/2年/5年

### 3.2 注目銘柄スクリーニング

| 項目 | 内容 |
|---|---|
| 入力データ | 日本経済新聞マーケット速報＋Yahoo!ニュース経済RSSの見出し（件数はスライダーで調整） |
| 処理 | 選択した1つ以上のAIモデルに見出し一覧を渡し、翌営業日以降に株価が動きそうな銘柄を注目度順にランキングさせる |
| 出力形式 | モデルごとにタブ分けし、順位・銘柄名・ティッカー・関連ニュース・注目理由・想定インパクト（表形式）＋全体の相場観 |
| タイムアウト | モデルごとに設定可能（30〜300秒）。応答が止まったモデルはタイムアウトメッセージに切り替え、他のモデルの表示をブロックしない |

免責: ニュース見出しのみに基づく速報的な参考情報であり、投資判断の根拠とするものではない旨を明記する。

### 3.3 ウォッチリスト一覧

- 期間切り替え（1日/1週/1カ月/6カ月/1年/2年）を全リスト共通で適用
- 🌐公開／🏘️コミュニティ／👤個人の3タブ（コミュニティ・個人はホーム画面で認証済みの場合のみ内容表示、未認証時は案内メッセージ）
- 各銘柄カード: 銘柄名（クリックで分析画面へ）、現在値・前日比、日付軸・価格帯付きチャート、メモ（自由記述、保存ボタンで永続化）、フォルダ（自由記述のグループ名、保存すると同名でグルーピング）
- フォルダはラジオボタンで切り替え表示。フォルダ未設定の銘柄は「その他」に集約

### 3.4 チャート学習

#### 3.4.1 テクニカル指標パターン（8種）

MAクロス（ゴールデンクロス）、RSI（売られすぎ）、MACDクロス、ボリンジャーバンド（下限タッチ）、一目均衡表（雲抜け）、ADX（トレンド強度）、OBV（出来高の裏付け）、VWAPのそれぞれについて、サンプルチャートと「どんな形が出たら買い/売りサインか」の解説を表示する。

#### 3.4.2 ローソク足パターン（7種）

陽の丸坊主、陰の丸坊主、十字線（同事線）、ハンマー、首吊り線、陽の包み足、陰の包み足について、サンプルのローソク足チャートと解説を表示する。

#### 3.4.3 該当銘柄の検出

各パターンについて、公開・個人・コミュニティの全ウォッチリスト銘柄（重複除去）を対象に、直近データでそのパターンに該当するかを判定し、該当する銘柄をボタンで一覧表示する。ウォッチリストが空、または該当銘柄がない場合はその旨のメッセージを表示する。クリックでその銘柄の分析画面に遷移する。

### 3.5 使い方

各画面の使い方・ウォッチリストの3階層の違い・AIモデルの違いと無料枠の注意点・免責事項を1画面にまとめた静的なガイド。各セクションから該当ページへのリンクを設置する。

### 3.6 条件スクリーニング

| 条件項目 | 単位・形式 |
|---|---|
| 市場 | 日本株／米国株（ラジオボタン） |
| 時価総額 | 億円／億ドル（下限・上限、0は上限なし扱い） |
| 売上高 | 同上 |
| PER | 倍（下限・上限） |
| ROE | %（下限・上限） |
| PBR | 倍（下限・上限） |
| 配当利回り | %（下限・上限） |
| セクター | 選択式（11業種＋指定なし） |

- 検索実行で該当件数と上位50件（時価総額降順）を表示。各結果カードに銘柄名・現在値・前日比・時価総額・PER・PBR・配当利回りと「分析」ボタンを表示
- 日本株の会社名は取得元（Yahoo!ファイナンス）が英語表記で返すため、自動的に日本語へ翻訳して表示する
- 該当銘柄がない場合は条件を緩めるよう案内する

## 4. 用語集

| 用語 | 説明 |
|---|---|
| 総合シグナル | 8つのテクニカル指標のうち買い/売りと判定された指標の比率から算出。75%以上で「買い」「売り」、過半数で「やや買い」「やや売り」、それ以外は「中立」 |
| 夜間PTS | 私設取引システム（Proprietary Trading System）における、証券取引所の取引時間外（17時〜翌6時）の株価。本アプリはジャパンネクスト証券J-Marketの値をYahoo!ファイナンス経由で表示する |
| 合言葉 | ウォッチリストの個人/コミュニティリストで、同じ名前を他人に使われないための簡易パスワード。強固な認証ではなく、なりすまし防止レベルの軽量な仕組み |
| 東証終値比 | 夜間PTSの取引値が、その日の東証終値に対してどれだけ変動したかを示す差分・騰落率 |

## 5. 非機能要件（概要）

詳細は[設計書.md](./設計書.md)7章を参照。要点のみ記載する。

- 外部データ取得はすべて失敗を許容し、取得不可時も他機能を止めない
- AI機能は無料枠のあるクラウドAPI（Gemini/Groq）またはローカルLLM（Ollama）を利用者が選択でき、料金は発生しない設計とする
- 個人情報の会員登録は行わず、ウォッチリストの区分けは合言葉ベースの軽量な仕組みに留める
- すべての分析結果・レポート・ランキングは参考情報であり、投資助言ではない旨を明示する
